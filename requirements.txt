# Project Requirements & Tech Stack

This document outlines the core technologies and dependencies required for the Health & Fitness Tracking application.

## 🟢 Core Tech Stack
- **Frontend**: React 19 (Functional Components + Hooks)
- **Build Tool**: Vite 8.x
- **Backend**: Node.js + Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Styling**: TailwindCSS 3.4+ (Custom "Clean UI" configuration)

## 🔵 Major Frontend Dependencies
- **react-router-dom**: Client-side routing for Dashboard and Forms.
- **react-toastify**: Real-time status notifications (Success/Error handling).
- **lucide-react**: Lightweight icon set for UI elements.
- **axios / native fetch**: Backend API communication.
- **context-api**: Centralized User Authentication & State persistence.

## 🟠 Major Backend Dependencies
- **express**: Core server framework.
- **mongoose**: Schema-based data modeling for MongoDB.
- **cors**: Cross-Origin Resource Sharing for Frontend-Backend communication.
- **jsonwebtoken (JWT)**: Secure user session management (Planned).

## 📊 Business Logic Modules
- **useScore Engine**: 60/40 Physical-Mental health scoring logic.
- **useCalories Engine**: Real-time metabolic and intake tracking.
- **Gym Volume Engine**: Repetition/reps based strength analysis.
- **Mental Wellness**: Integrated Yoga & Meditation tracking.
